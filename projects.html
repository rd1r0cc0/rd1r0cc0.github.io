PacMen Factory

  • Click on PacMan icon above to see the running code

  • 1 HTML file
  • 1 JavaScript file
  • 1 StyleSheet file

  • Starts with 2 Buttons: One to add PacMen and one to start the game.
  • You can create as many PacMen as you want (random position & velocity for each PacMan generated).
  • The app tracks position, velocity of each PacMan and bounces them off all four sides of the page.

Eye Movement

  • 1 JavaScript file
  • 1 StyleSheet file

Real-Time Bus Tracker

  • Click on Red Bus Marker icon above to see the running code

  • 1 HTML file
  • 1 JavaScript file
  • 1 StyleSheet file

  • Requires a Google Maps JavaScript API Key (mapsplatform.google.com)

  • Loads a Google map of Boston, pulls every 15 seconds transit bus data from MBTA.com for the bus route between MIT and Harvard, and adds markers to the map for each bus' current location on the route.
